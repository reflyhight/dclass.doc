
# 介绍
>Dclass是一套 Scratch 在线作品创作，管理系统。实现了scratch在线创作，保存，分享手机试玩，学生添加，用户注册等常用功能。解决目前少儿编程培训中小型机构无法开发自己的创作平台的痛点。




# 安装与使用
***Dclass全全负责安装部署，您的编程培训学校可以在完全不会开发的情况下使用***
